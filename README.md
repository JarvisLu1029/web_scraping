### 實作練習
**爬取任一購物網站需求如下:**  
1. 將程式碼打包成 function 並要有 物品名稱 參數
2. 該 function 在使用後要自動至網站爬取該物品的**所有商品內容**
3. 爬取內容至少要包含 (越多越好):
   - 商品名稱
   - 商品價格
   - 商品連結
   - 商品圖片連結
4. 如查詢不到該商品，要回傳錯誤訊息而不是讓程式報錯
5. 請將爬取到的資料輸出至 `<物品名稱>.json` 檔
6. 後續提交 `json檔` 以及 `py檔` 至講師信箱  
   標題格式: `[班級] 姓名-爬取的網站名稱`  
   Example: `[AIPE02] Jarvis-PChome`
